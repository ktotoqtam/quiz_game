тесты:
    создание тестов:
        /api/tests, метод-post
        структура запроса:
             {'title': название теста,
              'about': о чем тест,
              'questions': [
                {'type': тип вороса (ord - простой/ rad - с вариантом выбора/ check - чекбоксы),
                 'question':
                    если ord:
                        сам вопрос
                    если rad или check:
                        [сам вопрос[варианты выбора]],
                 'answer':
                    если ord или rad:
                        ответ
                    если check:
                        [ответы]]},
              'is_private': приватность теста (True/False),
              'user_id': id пользователя}
        возвращает ключ
    получение всех тестов:
        /api/tests, метод-get
        возвращает список с тестами
    получение теста по ключу:
        /api/tests/ключ, метод-get
        возвращает тест
    удаление теста по ключу:
        /api/tests/ключ, метод-delete
        ничего не возвращает


пользователи:
    создание пользователей:
        /api/users, метод-post
        структура запроса:
             {'name': имя пользователя,
              'about': о пользователе,
              'login': логин,
              'password': пароль
             }
        возвращает id
    получение всех пользователей:
        /api/users, метод-get
        возвращает список с пользователями
    получение пользователя по id:
        /api/users/id, метод-get
        возвращает пользователя
    удаление пользователя по id:
        /api/users/id, метод-delete
        ничего не возвращает

             {'user_id': id пользователя для которого предназначается ключ}
        возвращает ключ

        /api/telegram_keys/ключ, метод-get
        возвращает id пользователя
    удаление ключа по ключу:
